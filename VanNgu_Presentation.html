06Hearing speech over the machines

A microphone array,
not a single mic.

2–4 MEMS elements enable beamforming — steering toward the speaker and rejecting directional machine noise. A neural VAD trained on factory noise gates inference to real speech.

Key insight

Don't over-clean for the machine

The recognizer hears lightly-processed, noise-robust audio. Aggressive denoising — which adds artifacts that raise error — is reserved for human listening only.

On the DSP

Always-on, low power

Beamforming, VAD and feature extraction run on the Hexagon audio DSP — saving the CPU and the battery.

07Two tonal languages, in real time

Streaming recognition that
respects tone.

A streaming model emits words during speech and identifies the language inline — handling the code-switching workers actually use. Tone lives in a fragile pitch contour, so it gets special care.

Streaming

Finalizes in ~100 ms

Not a 30-second transcription window — a real-time transducer.

Tone-safe

Quantization-aware

Trained for low-bit hardware so near-homophone tones survive.

Measured

Tone confusion, tracked

Reported separately — never hidden inside an average error rate.

08The Industrial Knowledge Layer, made concrete

Terminology guaranteed by
mechanism — not luck.

Machine IDs, line numbers and SOP codes are detected and masked as typed tokens that pass through translation untouched, then restored in the target language. Mandated safety wording is forced by constrained decoding.

worked example
in   Kiểm tra máy số 3
mask Kiểm tra ⟨MACHINE_03⟩
nmt  检查 ⟨MACHINE_03⟩
out  检查3号机
token count in ≠ out → reject & ask again
09Safety is gated, not assumed

The worst failure is a confident, wrong safety instruction.

A fuzzy match must never turn “don't switch off” into “switch off.” So the match threshold is treated as a safety parameter, and the system refuses to guess.

Inversion guard

đừng tắt máy ≠ tắt máy

Negation and antonym pairs are kept provably distinct.

Fail-safe

When unsure, ask to repeat

Below a confidence margin, it stops rather than risk an inversion.

Zero-tolerance

Critical-tier gate

Any meaning inversion is a hard fail in evaluation.
